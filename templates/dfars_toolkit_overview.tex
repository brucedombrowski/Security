% ---- PURPOSE ----
\sectionheader{Purpose}

Automated security scanning toolkit that verifies software projects against
\textbf{NIST SP 800-171} (Protecting CUI), \textbf{NIST SP 800-53}, and
\textbf{FIPS 199/200}. Produces checksummed, traceable evidence with
file:line references for audit and assessment packages.
